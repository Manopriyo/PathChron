# PathChron 🎓

PathChron is a simple and clean YouTube Lecture Companion designed to help students take structured notes while watching lectures.

It eliminates distractions and improves revision efficiency by allowing timestamp-based note-taking and PDF export.

---
https://manopriyo.github.io/PathChron/
---

## 🚀 Features

- 🎥 Embed any YouTube lecture
- 📝 Auto-saving notes (LocalStorage based)
- ⏱️ Automatic timestamp insertion
- 🔗 Clickable timestamps (jump to that exact moment in video)
- 📄 Export notes as PDF
- 🌙 Dark mode support
- 💯 100% free & no login required

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- YouTube Iframe API
- jsPDF (for PDF export)
- LocalStorage (for auto-save)

---

## 📦 How It Works

1. Paste a YouTube lecture link.
2. The video loads inside the website.
3. Click **Add Timestamp** to automatically insert the current video time.
4. Click on timestamps in preview to jump back to that moment.
5. Download structured notes as a PDF (includes video title and URL).
